### Installation

Firstly, clone the latest version of the repo:
```sh
$ git clone https://github.com/didimoner/deeengine.git
```

Next install all packages and build the app:

```sh
$ cd deeengine
$ npm i
$ npm run buld
```

### Development

If you want to make your own changes, run webpack dev server:

```sh
$ npm run serve
```
